# Incidence Troubleshooter (Minimal)

Minimal project for the assignment: generate incidents, embed to Chroma, run a small Streamlit app with resolver + critic agents.

## Structure
- `data/` : generated incident files and chunks.
- `scripts/` : generation, preprocess, embed scripts.
- `app.py` (added later) : Streamlit UI.
- `.gitignore` : files to ignore in git.
- `requirements.txt` : Python dependencies.

## Quick commands
- Activate venv: `.venv\Scripts\activate`
- Install deps: `pip install -r requirements.txt`
- Run Streamlit (later): `streamlit run app.py`
